### Lunch it locally
1. Set up a python interpreter (e.g. Miniconda)
2. Install repo:
```
git clone https://github.com/spisakt/rl_lecture.git
cd rl_lecture
pip3 install -r requirements.txt
jupyter notebook
```
3. Present with [RISE](https://rise.readthedocs.io/en/stable/index.html).
